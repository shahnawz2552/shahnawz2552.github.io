03 — Projects

Things I Have Built

Selected work demonstrating my approach to solving real problems with code.

001

Colour Recognition via ML

Real-time colour detection pipeline that classifies 15+ colour categories from live webcam input with 92% accuracy. Built a custom training dataset using OpenCV, applied KNN classification on HSV-space pixel samples, and exposed results via a lightweight Python CLI. Handles variable lighting through adaptive histogram equalisation.

Python OpenCV Scikit-learn KNN NumPy

002

JWT Authentication REST API

Stateless authentication service built with PHP and MySQL. Implements RS256-signed JWT issuance, token refresh with rotation, and role-based access control across protected endpoints. Password hashing uses bcrypt with per-user salts. Designed with separation of concerns — controllers, middleware, and repository layers kept independent for testability.

PHP MySQL JWT REST API bcrypt RBAC

003

Sports Tournament Management System

Full-stack web application managing multi-sport tournaments — team registration, fixture generation (round-robin and knockout brackets), live score updates, and a public-facing results dashboard. Backend built in PHP with a normalised MySQL schema (8 tables, foreign-key constraints). Includes an admin panel with role separation between organiser and referee accounts.

PHP MySQL JavaScript Bootstrap HTML/CSS

004

Student PDF Marketplace

Peer-to-peer platform where students upload and download study materials (PDFs, notes, question papers). Features secure file upload with MIME-type validation, search and filter by subject/semester, user ratings, and a credit-based download system. Built on PHP/MySQL with PDO prepared statements throughout to prevent SQL injection.

PHP MySQL PDO File Upload Bootstrap

005

Utility Bill & Online Services Manager

Centralised dashboard for tracking household utility bills (electricity, water, internet) and recurring online service subscriptions. Users set due-date reminders, view monthly spend trends via Chart.js, and export reports as CSV. Built with a PHP backend, MySQL relational schema, and vanilla JS frontend — no frameworks, no build step.

PHP MySQL Chart.js JavaScript CSV Export
